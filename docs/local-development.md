# Local Development Instructions

## Prerequisites

* Node.js
* pnpm

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
pnpm install
```

## Running Locally

Start the development server:

```bash
pnpm dev
```

## Building for Production

Build the assets for deployment:

```bash
pnpm build
```

The output will be in the `dist` folder.
